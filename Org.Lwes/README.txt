The IoC support built into this API requires the CommonServiceLocator v1.0 found 
	at http://www.codeplex.com/CommonServiceLocator. You'll find links to your
	favorite IoC container among the links on that page.
	
	
The VSTS test projects require the mocking framework MOQ, it can be found 
	at http://code.google.com/p/moq/
