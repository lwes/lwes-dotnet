This directory is the drop target for the UpdateWixProjectWithVersion project. 

UpdateWixProjectWithVersion is a command line utility that keeps the LwesBinaryInstaller project up to date with the most recently built version of Org.Lwes.dll (the main LWES.net library). The utility is used in the Org.Lwes project's the post-build step.
